# CNN-for-Brain-Tumor-Detection-in-MRI-with-Grad-CAM-Interpretability

Description:
An advanced object detection system using YOLOv8-OBB for detecting small, oriented objects in aerial images. Trained on the DOTA dataset and validated on real-time video streams.

Key Features:

Oriented bounding box detection

Custom evaluation metrics

Real-time vehicle detection

Tech Stack: Python, YOLOv8, OpenCV, Deep Learning

## Dataset
The MRI brain tumor dataset is available on Kaggle:
https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection
